# Changelog

All notable changes to statanim will be documented here.

## [0.1.2] - 2026-06-02
### Fixed
- Bug fixes and stability improvements

## [0.1.1] - 2026-06-02
### Added
- Additional distribution visualisers

## [0.1.0] - 2026-06-01
### Added
- Initial release
- Distributions: Normal, Binomial, Poisson, Hypergeometric, Beta, Gamma, Chi-squared, Student-t, F
- Charts: BarChart3D, Histogram3D, BoxPlot3D, ViolinPlot3D, HeatMap3D, ScatterPlot3D, LinePlot3D
- Probability: SampleSpace3D, VennDiagram3D, ProbabilityTree3D, BayesBox3D
- Inference: HypothesisRegion3D, ConfidenceInterval3D, SamplingDistribution3D
- Regression: RegressionPlane3D, ScatterCloud3D
- Props: Card3D, Deck3D, Coin3D, Die3D, Spinner3D, Urn3D
- 6 built-in themes
